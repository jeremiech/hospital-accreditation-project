# Hospital Accreditation Backend

### Requirements

- node 18
- mongodb
- browser

### Installation

- `git clone https://github.com/6hislain/hospital-backend`
- `cd hospital-backend`
- `npm install`
- rename `.env.example` to `.env` and update configuration
- `npm start` or `npm dev`
